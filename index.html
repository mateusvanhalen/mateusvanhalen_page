Utripn


Click for Code





Utripn is an abbreviated journal app built in react.js that is friendly for users of ALL ages to share, recommend, and recap their trips of all shapes and sizes. Utripn solves the problem for bulky wordy journals and provides a quick and fun reference. Some add on features like picture uploading and a youtube reference video give some aesthetics and add more sensory reminders for the user.



React - Nutshell


Click for Code





Nutshell React is a single page react app that hosts a series of events, jokes, news, messages, and events for uses to store and track data. Nutshell served as a learning tool to effectivley utilize CRUD operations and session storage in React.js. This group sprint took roughly one week of class time and was our first attempt at a React group project.
