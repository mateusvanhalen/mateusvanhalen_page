Utripn

Utripn is an abbreviated journal app built in react.js that is friendly for users of ALL ages to share,recommend, and recap their trips of all shapes and sizes. Utripn solves the problem for bulky wordy journals and provides a quick and fun reference. Some add on features like picture uploading and a youtube reference video give some aesthetics and add more sensory reminders for the user.

React, CSS, JSON

Nutshell

Nutshell React is a single page react app that hosts a series of events, jokes, news, messages, and events for uses to store and track data. Nutshell served as a learning tool to effectivley utilize CRUD operations and session storage in React.js. This sprint took roughly one week of class time and was our first attempt at a React group project.

React, CSS, Group Project and first working React Application

Chore Tracker

Chore Tacker (v.1) is a beta version of a family friendly web application that lets families and users track their chores. The parent profiles are able to administer chores for that family and the childres profiles are able to view and sort. The app was built to solve the problem of "forgetful" chore isseus for the entire family. I would love to hear your feedback on Chore Tracker so far!

C#, Entity Framework, Identity Framework, MVC, SQL, Bootstrap
